# @tabnas/jsonic-cli

The `jsonic` command — a command-line interface for
[`@tabnas/jsonic`](https://github.com/tabnas/jsonic), a JSON parser that isn't
strict.

It reads relaxed-JSON source (from arguments, `--file`, or STDIN), parses it
with jsonic, merges the results, and prints standard JSON via
`JSON.stringify`.

## Install

```sh
npm install -g @tabnas/jsonic-cli
```

This package declares `@tabnas/jsonic` (and the optional `@tabnas/debug`) as
peer dependencies; npm 7+ installs them automatically.

## Usage

```sh
jsonic <args> [<source-text>]*
```

`<source-text>` is parsed into JSON. If omitted, source is read from STDIN.
Multiple sources are merged in precedence (highest first): right to left,
STDIN, then `--file`.

### Examples

```sh
# Basic
jsonic a:1
# => {"a":1}

# Indented output
jsonic -n a:1
# => {
#      "a": 1
#    }

# Merging arguments
jsonic a:b:1 a:c:2
# => {"a":{"b":1,"c":2}}

# Piping
echo a:1 | jsonic
# => {"a":1}

# Using a plugin (e.g. npm install @tabnas/csv)
jsonic -p csv -o plugin.csv.record.separators=^ "a,b^1,2"
# => [{"a":"1","b":"2"}]
```

### Arguments

| Argument | Description |
|---|---|
| `-` | Alias for STDIN. |
| `--file <file>` / `-f` | Load and parse `<file>`. |
| `--option <name=value>` / `-o` | Set option `<name>` (dotted path) to `<value>`. |
| `--nice` / `-n` | Print JSON indented over multiple lines. |
| `--meta <meta=value>` / `-m` | Set parse meta-data (dotted path). |
| `--plugin <require>` / `-p` | Load a plugin by module reference (name or path). |
| `--debug` / `-d` | Print abbreviated lex and parse logs (requires `@tabnas/debug`). |
| `--help` / `-h` | Print the help message. |

Output is generated by `JSON.stringify`; its `replacer` and `space` arguments
can be set with `-o JSON.replacer=...` and `-o JSON.space=...`.

### Debug

The `--debug` flag loads the [`@tabnas/debug`](https://github.com/tabnas/debug)
plugin to print a grammar description and a token-by-token trace:

```sh
jsonic -d -o plugin.debug.trace=true a:1
```

`@tabnas/debug` is a peer dependency of this package, so npm 7+ installs it
automatically alongside the CLI.

## BNF / grammar conversion

The BNF-to-grammar converter and its CLI live in a separate package,
[`@tabnas/bnf`](https://github.com/tabnas/abnf), exposed as the `tabnas-bnf`
command.
